# UE---Lighting_BP

## 项目简介

基于 Unreal Engine 蓝图开发的模块化灯光工具，旨在解决传统手动布光流程繁琐、迭代效率低的问题。通过 Spring Arm 机制设计，支持快速搭建与调整场景灯光。

## 功能特性

- 从 **Spring Arm** 出发，快速添加多个灯光
- 每个灯光可独立控制参数，支持设置：
  - 灯光类型（Point / Spot / Rect Light）
  - 颜色、强度、色温、衰减
- 通过**数据资产**设置预设灯光模型（如三点布光预设），灵活复用
- 通过蓝图灵活控制灯光数量、类型及位置
- 快速搭建场景灯光，大幅提升布光效率

## 项目结构

```
Content/
└── Light/
    ├── Bluepint/     # 蓝图核心文件
    ├── level/        # 场景文件与材质
    └── presets/      # 灯光预设数据资产
level_P.umap          # 主场景
```
## 使用方法

1. 将 `Content` 文件夹导入到你的 UE 项目中
2. 打开 `level_P.umap` 查看示例场景
3. 在蓝图中调用灯光组件，设置所需参数
4. 通过数据资产切换预设灯光方案
